Please follow these steps to use the color blender tool effectively: choose a valid color value. Enter two valid CSS color values in the chosen format. Select the number of midpoints you want to see. The palette will display the colors you entered, along with the requested number of midpoint colors and their corresponding values. Note that all numbers are rounded to the nearest integer. To apply a color value to the highlighted input, click on the corresponding square in the "waterfall" display. If needed, you can switch between different value formats, and the tool will automatically translate the existing values to the new format. To clear all values and colors, click the "Clear" button. Please note that this action does not change the current value format.
